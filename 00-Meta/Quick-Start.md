## 💡 Pro Tips

1. **Start messy, refine later** — Don't over-engineer the first week
2. **One insight per atomic note** — Short = linkable = useful
3. **Link immediately** — A note without links is a dead end
4. **Use %% comments** — Agent instructions invisible in reading mode
5. **Trust the weekly review** — 30 min/week > any plugin feature
