# 📝 Bắc Giang Content Template Framework — Mẫu content theo nền tảng

> [!SUMMARY] Bộ template viết sẵn cho từng nền tảng, ngành hàng và mục đích. Áp dụng đúng khung BG để nội dung "đúng chất".

## 1. TEMPLATE FACEBOOK POST (Bán nông sản)

```
[Hook tự hào quê hương] + [Lợi ích cụ thể] + [Cam kết chất lượng] + [CTA nhẹ]

"Vải thiều chín đỏ vườn nhà hôm nay ạ 🍒
Chín tới nơi, ngọt đúng vị. Đóng gói cẩn thận, guarantee tươi từ lá đến quả.
Ai muốn thử thì inbox em nhé, ship tận nơi ạ.
Số lượng có hạn nên anh/chị nào muốn mua sớm thì nhắn em trước nha."
```

## 2. TEMPLATE TIKTOK SHOP (Livestream/Selling)

```
[Các bác ơi] + [Lợi ích sản phẩm] + [Giá/Ưu đãi] + [CTA khẩn cấp] + [emoji]

"Các bác ơi, vải chín đỏ vườn nhà hôm nay nha 😍
Quả to đều, ngọt lịm luôn nè. Mua 2kg free ship cả nước ❤️
Nhanh ghé tớ lựa nha, hàng có hạn thôi ạ! 🏃‍♂️"
```

## 3. TEMPLATE COMMENT GROUP FB (Góp ý/Tham gia)

```
[Chào cộng đồng] + [Lập trường] + [Góp ý xây dựng] + [Kết thúc kính trọng]

"Thưa cả nhà, em cũng đồng ý với ý kiến này ạ.
Nhưng mà bác có nghĩ là...? Em thấy bên mình đã làm tốt rồi, nếu thêm phần này thì càng hoàn thiện hơn ạ."
```

## 4. TEMPLATE COMMENT FB (Tương tác khách hàng)

```
[Lời chào] + [Trả lời câu hỏi] + [Offer] + [CTA]

"Dạ, em chào anh/chị ạ. Vải năm nay chín đều lắm ạ.
Anh/chị muốn mua bao nhiêu thì em báo giá cụ thể nhé.
Ship tận nơi, guarantee tươi từ lá đến quả nha ❤️"
```

## 5. TEMPLATE INSTAGRAM CAPTION

```
[Ngắn gọn] + [Hình ảnh cụ thể] + [Emoji tối giản]

"Sáng nay ra chợ sớm, mua được mấy củ măng tươi.
Về kho với cá thì ngon hết nước chấm luôn 🐟"
```

## 6. TEMPLATE THREADS/TWITTER

```
[Đi thẳng vấn đề] + [Tự hào BG] + [ạ cuối câu] + [Emoji]

"Vải thiều Lục Ngạn năm nay ngon lắm ạ. Chín đỏ cả đồi 🍒
Ai ở Bắc Ninh/Bắc Giang thì thử nhé!"
```

## 7. TEMPLATE BĐS (NOXH/KCN)

```
[Vị trí gần KCN] + [Giá] + [Tiện ích] + [CTA nhắn tin]

"1pn-2pn cho thuê gần KCN Vân Trung, giá từ 3.3tr/tháng 😍
Phòng đầy đủ nội thất, ra trung tâm thương mại chưa đến 5p đi bộ.
Nhắn e để nhận thông tin chi tiết nha ❤️"
```

## 8. TEMPLATE EDUCATIONAL CONTENT (TikTok)

```
[Đặt vấn đề] + [Tips kỹ thuật] + [Kết quả thực tế] + [CTA follow]

"Nhiều người khoanh vải thiều nhưng không biết điều này 🤔
Thưa bác, chỉ cần cắt ngọn cây, để 3 ngày rồi mới ghép...
Cây cho năng suất cao hơn hẳn ạ! Follow em mỗi ngày để học thêm kỹ thuật trồng trọt nha ❤️"
```

| Livestream | Năng lượng + thân mật | "các bác ơi", "nhanh ghé", emoji ❤️😍 |
| Group FB | Cộng đồng + xây dựng | "cả nhà", "thưa", "em xin phép" |

| Educational | Thực tế + kỹ thuật | "nhiều người chưa biết", "tips", "kết quả" |
